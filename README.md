# Kami North's Blackjack 2
This project has all the content, including spoilers. If you are an active player, **please do not browse around this repo.**

Not sure where to start looking? Check out the [Notebook!](https://github.com/Nutik-Wulf/Kami-North-Blackjack-2/blob/e6dfc090c75a6cfedd04b329570792a9ba91138e/Guidebooks/0-Notebook.pdf) It has good documentation and information about the game, as well as how it differs from traditional Blackjack.

Contained within this repo is primarily all custom assets currently in use for my custom Blackjack table, which is hosted on [Tabletop Simulator](https://store.steampowered.com/app/286160/Tabletop_Simulator/). Any of my scripts used within the game's Lua engine is also included in this repository, as well as a standalone python script contributed by good friend and active player [MiniChris](https://github.com/Mini-Chris). Current designs for elements are made by [Hanzaki of Hoshimadara Lab](https://hoshimadara.tumblr.com/), and Everything Else is handmade myself.

Almost all artwork was made using [Aseprite](https://www.aseprite.org/) and [Krita](https://krita.org/en/).
The background renders were made using [Bryce 7](https://www.daz3d.com/bryce-7-pro).
Custom models were made using [Blender](https://www.blender.org/).
